# ParkEase Frontend

## Project Description
ParkEase is a web-based software solution designed to digitize the day-to-day operations of a public parking facility that also offers tyre clinic and battery hire services. The system will support accurate record keeping, automated fee calculation, receipt generation, and daily revenue reporting. This repo will consist of the front-end only.
